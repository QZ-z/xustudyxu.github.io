## Spring5 框架概述

1. Spring是**轻量级的开源的JavaEE框架**。
2. Spring可以解决企业应用开发的复杂性。
3. Spring有两个核心部分:**IOC和Aop**。
   - <font color=#DC4040 size=4 face="黑体">IOC:控制反转,把创建对象过程交给spring进行。</font>
   - <font color=#DC4040 size=4 face="黑体">Aop:面向切面,不修改源代码进行功能增强。</font>
4. Spring特点
   1. 方便解耦，简化开发
   2. AOP编程支持
   3. 方便程序测试
   4. 方便和其他框架进行整合
   5. 方便进行事务操作
   6. 降低API开发难度
